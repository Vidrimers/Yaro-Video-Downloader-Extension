# 🎬 Yaro Video Downloader

Мощное расширение для Chrome/Opera/Edge, которое позволяет скачивать видео с любых сайтов, включая защищённые потоковые видео.

## ✨ Возможности

- 📥 **Прямое скачивание** - скачивайте обычные видео одним кликом
- 🔒 **Защищённые видео** - перехватывайте и собирайте сегменты из MediaSource API
- 🎯 **Автоопределение** - автоматически находит все видео на странице
- ⏺️ **Управление записью** - начинайте и останавливайте запись сегментов
- 📊 **Статистика** - отслеживайте количество сегментов и время записи
- 🎨 **Красивый UI** - современный интерфейс с градиентами
- 🚀 **Быстрая работа** - оптимизированная архитектура
- 🎬 **mp4box.js** - правильная сборка MP4 с поддержкой треков и метаданных
- ℹ️ **Информация о видео** - просмотр codec, разрешения, треков

## 🚀 Установка

### Способ 1: Из исходников (для разработчиков)

1. Скачайте или клонируйте этот репозиторий
2. Откройте Chrome/Opera/Edge
3. Перейдите в `chrome://extensions/` (или `opera://extensions/`, `edge://extensions/`)
4. Включите "Режим разработчика" (Developer mode)
5. Нажмите "Загрузить распакованное расширение" (Load unpacked)
6. Выберите папку с расширением

### Способ 2: Из .crx файла (скоро)

Упакованная версия будет доступна позже.

## 📖 Как использовать

1. **Откройте страницу с видео** - расширение автоматически начнёт искать видео
2. **Кликните на иконку расширения** - откроется popup с найденными видео
3. **Выберите действие**:
   - Для обычных видео: нажмите кнопку "Скачать" рядом с видео
   - Для защищённых видео: нажмите "Начать запись", воспроизведите видео, затем "Скачать сегменты"

### Для защищённых видео (HLS, DASH, MSE)

1. Нажмите **"▶ Начать запись"** перед воспроизведением видео
2. Запустите воспроизведение видео на сайте
3. Дождитесь загрузки нужной части (следите за счётчиком сегментов)
4. Нажмите **"⏹ Остановить"**
5. Нажмите **"💾 Скачать сегменты"**

## 🛠️ Технологии

- **Manifest V3** - современный стандарт расширений Chrome
- **MediaSource API** - перехват защищённых видеопотоков
- **MutationObserver** - отслеживание динамически добавляемых видео
- **Chrome Extensions API** - downloads, scripting, storage
- **mp4box.js** - правильная сборка MP4 файлов из сегментов
- **MP4Builder** - модуль с fallback на простую конкатенацию

## 🔧 Разработка

### Архитектура

Расширение состоит из трёх основных компонентов:

1. **Injector** (`injector.js`) - внедряется в контекст страницы и перехватывает:
   - `MediaSource.addSourceBuffer()` - для определения MIME типа
   - `SourceBuffer.appendBuffer()` - для перехвата сегментов
   - Обычные `<video>` элементы на странице

2. **Content Script** (`content.js`) - мост между injector и background:
   - Получает сообщения от injector через `window.postMessage`
   - Отправляет данные в background через `chrome.runtime.sendMessage`

3. **Background Service Worker** (`background.js`) - управляет данными:
   - Хранит сегменты и метаданные по табам
   - Обрабатывает запросы на скачивание
   - Управляет состоянием записи

4. **Popup** (`popup.html` + `popup.js`) - пользовательский интерфейс:
   - Отображает найденные видео
   - Показывает статистику
   - Управляет записью и скачиванием
   - Показывает информацию о MP4 (треки, codec, разрешение)

5. **MP4Builder** (`mp4-builder.js`) - модуль сборки MP4:
   - Использует mp4box.js для правильной сборки
   - Fallback на простую конкатенацию если mp4box не работает
   - Парсинг информации о треках и метаданных

### Логирование

Все компоненты логируют свои действия в консоль:
- Injector: консоль страницы (F12)
- Content Script: консоль страницы (F12)
- Background: консоль расширения (`chrome://extensions/` → Details → Inspect views: service worker)
- Popup: консоль popup (ПКМ на popup → Inspect)

## 🐛 Известные ограничения

- Не все сайты используют MediaSource API (некоторые используют WebRTC или другие технологии)
- Некоторые сайты могут блокировать перехват через Content Security Policy
- mp4box.js может не работать со всеми форматами сегментов (в этом случае используется простая конкатенация)
- DRM-защищённые видео (Widevine, PlayReady) не поддерживаются
